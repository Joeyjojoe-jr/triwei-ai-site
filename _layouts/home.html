Operating charter

Playful, not careless.

  • Educational exhibits over personal commentary.
  • Source and method boundaries remain visible.
  • No visitor data leaves the browser.
  • Reduced-motion and keyboard access are first-class.
  • The live news system stays outside the public lobby.
Read the methodology →
Maintenance hatch

This prototype is intentionally static. Interactive state is stored only in the visitor's browser and can be cleared from the Machine Room.
