# TriWei Curiosity Mall v0.1

## Purpose

Create a project-first, identity-minimal interactive AI learning laboratory on the existing Jekyll and GitHub Pages stack.

## Non-negotiable boundaries

- No personal biography, photograph, identity profile, or personal opinion page.
- No merge to `main` or production deployment without explicit approval.
- No paid API, database, user account, tracking network, or production AI inference.
- Automated news output is an Observatory instrument, not the homepage identity.
- External links and machine labels do not indicate endorsement.
- Every spatial or animated interaction requires a conventional accessible route.

## v0.1 locations

1. Atrium — stable project introduction and directory.
2. AI Signal Observatory — conservative presentation of automated collection output.
3. Systems Workshop — browser-only teaching experiments.
4. Machine Room — architecture, privacy, controls, and limitations.
5. Methodology — editorial, technical, privacy, and professional boundaries.

## Deferred locations

- Prompt Planetarium
- Inference Greenhouse
- Systems Arcade
- Small Machines Museum

## Release gates before merge

- Jekyll build succeeds.
- Internal links resolve.
- Mobile and keyboard navigation are manually reviewed.
- Reduced-motion behavior remains functional.
- No personal information or opinion content is introduced.
- No external runtime dependency is added.
- Automated news language remains neutral and caveated.
