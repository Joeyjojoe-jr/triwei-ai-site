Use this page as your default re-entry checklist whenever you return to the hub.

## 1. Pick a Focus

Choose one topic from the Knowledge Map and define a single question you want answered today.

## 2. Run One Experiment

Open the Experiments section, copy an existing setup, and change only one variable.

## 3. Capture One Note

Write a short blog note in `/blog/` with what changed and what is still unclear.

## 4. Link Related Pages

At the bottom of each page, add links to connected guides or knowledge topics.
